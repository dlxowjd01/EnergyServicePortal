* API 공통

- 파라메터에서 '지역구분', '사이트명' 삭제
- 파라메터에서 '사이트ID'를 '장치ID'로 변경



* 충전계획량 리스트 조회 API

- 결과값에서 '충전계획량' 수집여부 -> 인코어드에서 수집방안 제공한다고 함 (by 김현탁 PM)



* 방전계획량 리스트 조회 API

- 결과값에서 '방전계획량' 수집여부 -> 인코어드에서 수집방안 제공한다고 함 (by 김현탁 PM)



* 발전량 리스트 조회 API

- 결과값에서 '발전량(예측)' 삭제



* BMS 장치 리스트 조회 API

- 결과값에서 'SOC최대', 'SOC최소', 'C-rate' 삭제



* PV 장치 리스트 조회 API

- 결과값에서 '수평일사량', '경사일사량', '오늘예측발전량' 삭제



* 장치 알람 리스트 조회 API

- 결과값에서 '알람상태', '조치여부' 삭제



* 실시간알람 API 연동 추가

- EWP 서비스포털에서 실시간알람 수신을 위한 API 제공예정. 
- 제공시 알람발생시 실시간으로 해당 API(URL호출방식)를 Call하여 알람정보 전달 (PMGrow -> EWP서비스포털)
